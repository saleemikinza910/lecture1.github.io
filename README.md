# Personal Portfolio Website

## 📌 Project Overview

This project is a **personal portfolio website** created using HTML and CSS.
It represents my personality, interests, and goals as a Computer Science student.

The website highlights:

* My personal story
* Favourite novels (Urdu & English)
* Series I watch
* My career goals
* Additional personal details

---

## 🎯 Objective

The main objective of this project is to:

* Practice **HTML structure**
* Apply **CSS styling and layout techniques**
* Build a **responsive and visually appealing webpage**
* Present personal information in a creative way

---

## 🛠️ Technologies Used

* HTML5
* CSS3

---

## ✨ Features

* Structured layout using semantic HTML elements
* Styled sections with custom colors and design
* Responsive design for different screen sizes
* Organized content with lists, cards, and sections
